Analytics Information for lpopsic.github.io

2024-04-02

By PopsicL

General Information

Starting April 1st, almost all websites in lpopsic.github.io will be tracked using Google Analytics. We will track general info, whether it is a country, or where the user is on a page, but not any sensitive info. Every page path will be tracked, with the exception of some specific websites. There are ways on how to bypass this. Scroll down the page for more information about this new change.

What will you track?

We will track users and new users entering the website; the average engagement time; new users by first user channel groups, sources, mediums and campaigns; sessions by session channel groups, sources, mediums and campaigns; users by country; views by page title and screen class and event counts. lpopsic.github.io and Google Analytics will NEVER track any sensitive info such as IP, your internet service provider (ISP), address, etc.

Where will you track?

We will track EVERY PAGE PATH (excluding the error and random stuff days' page paths) in lpopsic.github.io.

Why will you track?

The data collected by lpopsic.github.io will be used for statistic purposes.

Is my data going to be affected?

lpopsic.github.io will NEVER collect and sell your data to third-party advertisers. Again, we only track general info, and of course, lpopsic.github.io and Google Analytics will NEVER track any sensitive info such as IP, your internet service provider (ISP), address, etc.

Is there a loophole for this?

Suprisingly, yeah. There are 2 ways of doing this: 1. Try downloading the website from GitHub (https://github.com/LPopsic/lpopsic.github.io) and try manually editing the Google script tag. It is just after the head tag. 2. You can try using a VPN to bypass the data collection, but it might collect the data from the VPN too. (as far as I know)
